## Project Description
[Waste Classifier](https://waste-classifier-dvnkd7jhsqmjiyvbmkaqpx.streamlit.app/) is an AI-powered application built with Python and Streamlit that classifies images of waste into different types (Plastic, Organic, Paper, Metal, E-waste). The app also provides recycling tips and highlights the health, environmental, and climatic hazards associated with each type of waste. This project promotes sustainable development and responsible waste management practices.

---

## Features
- Classifies waste into categories: Plastic, Organic, Paper, Metal, E-waste.  
- Displays a relevant emoji along with the classification.  
- Provides recycling tips for each type of waste.  
- Shows **Health, Environmental, and Climatic hazards** in interactive colored cards.  
- Sidebar includes project description, tools used, and SDG impact.  
- User-friendly interface built using Streamlit.
